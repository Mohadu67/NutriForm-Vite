2. Données collectées

2.1 Données de santé (via Health Connect / HealthKit)

⚠️ Données sensibles (Article 9 RGPD)
Les données de santé sont des données sensibles nécessitant votre consentement explicite.
Catégorie Données collectées Source
Activité physique Nombre de pas, distance parcourue, séances d'exercice, calories brûlées (actives et totales) Health Connect (Android) / HealthKit (iOS)
Mesures corporelles Poids, taille, IMC (calculé) Health Connect / HealthKit
Données physiologiques Fréquence cardiaque (min, max, moyenne) Health Connect / HealthKit
Cycle menstruel Périodes menstruelles, flux (si vous choisissez de partager) Health Connect / HealthKit

2.2 Données de profil et d'identification

  • Identité : prénom, pseudo, adresse email
  • Photo de profil : prise avec la caméra ou depuis votre galerie
  • Informations fitness : âge, niveau de forme physique (débutant/intermédiaire/avancé), types d'entraînement préférés
  • Biographie : texte libre de présentation (optionnel)
  • Authentification : via email/mot de passe, Google Sign-In ou Apple Sign-In

2.3 Données de géolocalisation

  • Position GPS : latitude, longitude (précision à environ 10 mètres)
  • Adresse : ville, code postal (obtenu via reverse geocoding)
  • Finalité : matching social basé sur la proximité géographique

Important : Votre position GPS exacte est stockée sur nos serveurs pour permettre le matching par distance. Elle n'est jamais affichée aux autres utilisateurs (seule la distance approximative est visible, ex: "à 5 km").

2.4 Données de matching et préférences sociales

  • Disponibilités : créneaux horaires préférés (matin/après-midi/soir) pour chaque jour de la semaine
  • Critères de matching :
    • Distance maximale acceptée (rayon de recherche)
    • Niveaux de forme physique préférés des partenaires
    • Types d'entraînement préférés
    • Tranche d'âge préférée (min-max)
  • Actions sociales : likes, rejets, matches mutuels, blocages
  • Visibilité du profil : choix d'activer/désactiver le matching

2.5 Messages et communications

  • Chat P2P : messages texte, photos, vidéos, documents avec vos matches
  • Partages spéciaux :
    • Partage de localisation temporaire (expire après 1h)
    • Invitations à des séances d'entraînement
    • Partage de résumés de séances
  • Chatbot IA : conversations avec notre assistant virtuel pour le support et les conseils fitness
  • Support client : historique d'échanges avec notre équipe

2.6 Contenu créé

  • Recettes : titre, description, ingrédients, instructions, photos, informations nutritionnelles, temps de préparation/cuisson
  • Programmes d'entraînement : exercices, séries, répétitions, poids utilisés
  • Historique de séances : durée, calories brûlées, exercices effectués, notes personnelles
  • Likes et sauvegardes : recettes et programmes que vous aimez

2.7 Données techniques et de connexion

  • Token d'appareil : identifiant unique pour les notifications push (via Expo Push Notifications)
  • Système d'exploitation : iOS ou Android et version
  • Modèle d'appareil : pour optimiser l'affichage
  • Adresse IP : lors de la connexion aux serveurs
  • Logs de connexion : dates et heures d'utilisation de l'application
  • Données d'utilisation : fonctionnalités utilisées, temps passé, parcours dans l'application

2.8 Système de récompenses

  • Points XP : points gagnés via vos activités
  • Historique de rédemption : récompenses échangées chez nos partenaires
  • Éligibilité : suivi de votre progression

4. Utilisation de la caméra (Permission CAMERA)

L'application demande l'autorisation d'accéder à votre caméra uniquement lorsque vous souhaitez :

  • Prendre une photo de profil
  • Photographier vos repas ou recettes
  • Envoyer des photos dans le chat avec vos matches
Garanties de confidentialité :
  • Les photos sont prises uniquement à votre demande explicite via un bouton dans l'interface
  • L'application n'accède JAMAIS à votre caméra en arrière-plan
  • Aucune photo n'est prise ou uploadée sans votre action directe
  • Vous pouvez refuser cette permission : certaines fonctionnalités (photo de profil personnalisée, partage de photos) ne seront pas disponibles

5. Fonctionnalité de matching social

5.1 Comment fonctionne le matching

Harmonith propose une fonctionnalité de matching social pour vous connecter avec d'autres passionnés de fitness près de chez vous. Cette fonctionnalité est entièrement optionnelle.

5.2 Données visibles par les autres utilisateurs

Lorsque vous activez le matching, votre profil devient visible aux autres utilisateurs selon vos critères :

  • Photo de profil
  • Prénom ou pseudo
  • Âge
  • Bio (si renseignée)
  • Niveau de forme physique
  • Types d'entraînement préférés
  • Distance approximative (ex: "à 5 km") - JAMAIS votre position exacte
  • Disponibilités générales (créneaux horaires)
Ce qui n'est JAMAIS partagé :
  • Votre position GPS exacte (latitude/longitude)
  • Votre adresse précise
  • Votre nom de famille
  • Votre email
  • Vos données de santé (Health Connect)

5.3 Actions de matching

  • Like : manifester votre intérêt pour un profil
  • Reject : passer au profil suivant (vous ne verrez plus ce profil)
  • Match mutuel : quand deux personnes se likent, vous pouvez commencer à discuter
  • Block : bloquer un utilisateur (il ne pourra plus voir votre profil ni vous contacter)

5.4 Désactivation du matching

Vous pouvez désactiver le matching à tout moment dans les paramètres. Votre profil ne sera plus visible par les autres utilisateurs, mais vos matches existants et conversations seront conservés.

7. Sécurité des données

Nous mettons en œuvre des mesures de sécurité techniques et organisationnelles robustes pour protéger vos données :

7.1 Mesures techniques

  • Chiffrement en transit : HTTPS/TLS 1.3 pour toutes les communications
  • Chiffrement au repos : données chiffrées dans les bases de données
  • Stockage sécurisé sur appareil : tokens stockés dans le Keychain (iOS) / Keystore (Android)
  • Tokens JWT : authentification sécurisée avec expiration et refresh automatique
  • Hachage des mots de passe : bcrypt avec salt unique
  • Protection contre les attaques : rate limiting, protection CSRF/XSS
  • Logs d'accès : traçabilité des accès aux données sensibles

7.2 Mesures organisationnelles

  • Principe du moindre privilège : accès limité aux données selon les besoins
  • Formation du personnel : sensibilisation RGPD et sécurité
  • Contrôle d'accès : authentification forte pour les administrateurs
  • Audits réguliers : revue de sécurité et tests de pénétration
  • Plan de continuité : sauvegardes régulières, plan de reprise d'activité

7.3 En cas de violation de données

Si une violation de données personnelles est détectée, nous nous engageons à :

  • Notifier la CNIL dans les 72 heures (si résidence en France/UE)
  • Vous informer directement si la violation présente un risque élevé pour vos droits
  • Prendre les mesures pour limiter l'impact et empêcher de nouvelles violations

9. Vos droits RGPD

Conformément au RGPD, vous disposez des droits suivants sur vos données personnelles :

9.1 Droit d'accès (Article 15)

Vous pouvez obtenir une copie de toutes vos données personnelles que nous détenons.
Comment : Menu Paramètres → Confidentialité → Télécharger mes données, ou par email.

9.2 Droit de rectification (Article 16)

Vous pouvez corriger des données inexactes ou incomplètes.
Comment : Menu Paramètres → Modifier le profil, ou contactez-nous.

9.3 Droit à l'effacement / "Droit à l'oubli" (Article 17)

Vous pouvez demander la suppression de votre compte et de vos données personnelles.
Comment : Menu Paramètres → Compte → Supprimer mon compte.

9.4 Droit à la limitation du traitement (Article 18)

Vous pouvez demander de "geler" temporairement le traitement de vos données (par exemple pendant une contestation).
Comment : Contactez-nous à contact.harmonith@gmail.com.

9.5 Droit à la portabilité (Article 20)

Vous pouvez recevoir vos données dans un format structuré (JSON) pour les transférer à un autre service.
Comment : Menu Paramètres → Confidentialité → Exporter mes données (format JSON).

9.6 Droit d'opposition (Article 21)

Vous pouvez vous opposer au traitement de vos données pour des raisons tenant à votre situation particulière.
Exemples : S'opposer à l'utilisation de vos données à des fins d'analytics, marketing.
Comment : Contactez-nous à contact.harmonith@gmail.com.

9.7 Retrait du consentement (Article 7)

Lorsque le traitement repose sur votre consentement, vous pouvez le retirer à tout moment :

  • Health Connect / HealthKit : Paramètres téléphone → Applications → Harmonith → Autorisations → Révoquer
  • Géolocalisation : Paramètres téléphone → Localisation → Harmonith → Jamais
  • Notifications push : Paramètres téléphone → Notifications → Harmonith → Désactiver
  • Matching social : Menu Harmonith → Paramètres → Matching → Désactiver ma visibilité

9.8 Droit de ne pas faire l'objet d'une décision automatisée (Article 22)

Harmonith n'utilise pas de prise de décision entièrement automatisée produisant des effets juridiques ou vous affectant significativement. Le matching repose sur vos critères et actions volontaires (like/reject).

9.9 Délai de réponse

Nous nous engageons à répondre à vos demandes dans un délai de 1 mois (extensible à 3 mois si complexe, avec justification).

10. Données des mineurs

Âge minimum : 13 ans
Harmonith n'est pas destiné aux enfants de moins de 13 ans. Nous ne collectons pas sciemment de données personnelles d'enfants de moins de 13 ans.

Si vous avez connaissance qu'un enfant de moins de 13 ans nous a fourni des données personnelles, merci de nous contacter immédiatement à contact.harmonith@gmail.com. Nous supprimerons ces données.

Pour les mineurs de 13 à 15 ans (en France) : Conformément à la loi, l'inscription nécessite l'autorisation d'un titulaire de l'autorité parentale.
